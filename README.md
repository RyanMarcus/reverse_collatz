# Reverse Collatz Calculator

Built with this Rust:

    rustc 1.0.0-dev

And this cargo:

    cargo 0.0.1-pre-nightly (3f74d7e 2015-02-14) (built 2015-02-14)

To compile an executable, do:

    cargo build --release

To just run it, do:

    cargo run 6 # where 6 is the sequence length
